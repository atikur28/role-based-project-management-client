# Role-Based Project Management Frontend

## [Project Management Frontend Live Url](https://project-management-client-orpin.vercel.app)

## Description

This is the frontend for a Role-Based Admin & Project Management System. Users can register only via invite tokens. Admins have full control over user roles and project management.

Key features:

- Invite-based registration UI
- Role-based access & UI rendering
- Auth-protected routes
- User & Project management for ADMIN
- Loading & error handling
- Optimistic UI updates where applicable

## Tech Stack

- React
- TypeScript
- Redux Toolkit for state management
- Tailwind CSS for custom design
- Axios for API calls
- React Router for routing
